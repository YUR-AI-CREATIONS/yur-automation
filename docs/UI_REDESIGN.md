## Implementation Details

### CSS Features
- **Custom Tailwind config** with extended colors
- **Theme-specific variables** via body class (`theme-dark`, `theme-neon`, etc.)
- **Smooth transitions** (0.3s ease on all color/border changes)
- **Hover effects** with shadow glow on cards
- **Responsive grid** with `grid-cols-1 md:grid-cols-3`

### JavaScript Features
- **localStorage** for persistent theme selection
- **Real-time DOM updates** via class switching
- **No page reload** needed for theme change
- **Event listener** on theme selector

### CSS Classes Used
```
.accent           — Primary accent color
.accent-secondary — Secondary accent (neon only)
.card             — Card styling
.btn              — Button styling
body.theme-*      — Theme context
```

## Usage

### For Developers
The UI pages are generated by `core_ui.py`:

```python
from src.spokes.core_ui import generate_core_home_page, generate_loop_page

# In your Flask/FastAPI server:
@app.get("/ui", response_class=HTMLResponse)
async def ui_home():
    return generate_core_home_page()

@app.get("/ui/loop", response_class=HTMLResponse)
async def ui_loop():
    return generate_loop_page()
```

### For Users
1. **Visit** `/ui` or `/ui/loop`
2. **Click theme selector** (top-right corner)
3. **Choose your theme** — preference is saved
4. **Enjoy!** Smooth transitions and vivid colors
